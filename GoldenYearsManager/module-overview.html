养老院人员管理系统

基于Vue.js的前端框架，提供全面的养老院人员管理解决方案

认证模块

  • 登录页面（RSA加密）
  • Token管理
  • 自动刷新Token

用户管理

  • 用户列表（分页/搜索）
  • 用户详情页
  • 角色权限控制

人员管理

  • 人员分类筛选
  • 健康档案管理
  • 照片上传功能

护理计划

  • 计划列表（分页）
  • 计划接取/转交
  • 任务分配面板

护理记录

  • 记录时间轴视图
  • 快速录入表单
  • 修改历史对比

健康数据

  • 图表展示趋势
  • 健康档案编辑
  • 数据导出功能

提醒管理

  • 提醒列表分类
  • 周期提醒设置
  • 实时通知组件

技术栈

  • UI框架: Element UI / Vuetify
  • 可视化: ECharts / Chart.js
  • 路由管理: vue-router
  • 状态管理: vuex
  • HTTP请求: axios
  • 安全: jsencrypt (RSA加密)
  • 日期处理: dayjs

关键交互流程示例

护理计划接取流程

  1. 护理工在"可接取计划"列表点击"接取"按钮
  2. 调用API: POST /care_plan/{plan_id}/accept
  3. 成功后自动刷新列表，计划移入"我的计划"

健康数据录入流程

  1. 在人员详情页点击"添加健康记录"
  2. 填写表单并提交
  3. 自动更新图表和档案页
